# Portfólio — Jean Carlo Stein da Fonseca

Este projeto foi desenvolvido como parte da atividade prática extra do curso Introdução à Programação Front-End e Back-End (IP 20h A) do SENAI/SC LAB365. A proposta consiste na criação de uma landing page funcional destinada à apresentação de um portfólio pessoal, reunindo projetos já desenvolvidos, criados especificamente para o curso ou fictícios. O objetivo principal é consolidar, em um único espaço digital, informações relevantes sobre minha formação, habilidades técnicas e comportamentais, além de disponibilizar detalhes sobre projetos que demonstram meu aprendizado e evolução na área de programação. Esta página também serve como um instrumento inicial de apresentação profissional, podendo ser ampliada e atualizada ao longo da minha trajetória na área de tecnologia.

A landing page foi construída utilizando HTML, CSS e JavaScript, atendendo aos critérios de utilização de linguagens definidos no edital da atividade. Além disso, a estrutura foi organizada de forma semântica, responsiva e acessível, buscando entregar uma navegação clara, fluida e adaptada a diferentes tamanhos de tela. A página conta com seções bem definidas, um menu dinâmico com comportamento interativo via JavaScript, além de estilização cuidadosa para garantir boa legibilidade, contraste adequado e uma experiência visual coerente com o tema tecnológico.

## 📌 Objetivos do Projeto

- Criar um portfólio digital profissional alinhado à proposta da atividade prática extra.
- Consolidar habilidades e conhecimentos adquiridos no curso, incluindo HTML, CSS, JavaScript e boas práticas de desenvolvimento.
- Disponibilizar um espaço centralizado com informações pessoais, habilidades, projetos (reais ou fictícios) e meios de contato.
- Produzir um material que possa ser continuamente atualizado e melhorado ao longo da minha formação.

## 📂 Estrutura da Página

### 1. Hero / Apresentação

Inclui meu nome, título profissional e uma breve descrição do propósito do portfólio.

### 2. Sobre Mim

Apresenta informações pessoais, minha trajetória, interesses e experiências na área de desenvolvimento web.

### 3. Projetos

Lista de projetos concluídos ou fictícios, cada um contendo:

- Nome
- Descrição
- Tecnologias utilizadas
- Link para demonstração (quando aplicável)
- Link para o repositório (caso exista)

### 4. Habilidades

Inclui habilidades técnicas (HTML, CSS, JavaScript) e habilidades comportamentais (soft skills).

### 5. Contato

Apresenta meu e-mail, telefone, localização e links para redes sociais profissionais.

## 🛠 Tecnologias Utilizadas

- HTML5 — estrutura da página
- CSS3 — estilização, responsividade e layout moderno
- JavaScript — comportamento do menu mobile e interatividade
- Flexbox / Media Queries — responsividade
- Boas práticas de acessibilidade — atributos ARIA, textos descritivos e contraste adequado

## 📁 Como Executar o Projeto

1. Faça o download do arquivo `.zip` contendo todos os arquivos.
2. Extraia o conteúdo em uma pasta do seu computador.
3. Abra o arquivo index.html em qualquer navegador moderno.
4. Certifique-se de manter a estrutura de pastas, incluindo imagens e arquivos CSS/JS, para que tudo funcione corretamente.
